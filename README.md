# Mood Tracker Backend

Backend REST API built with Node.js, TypeScript, Express, and Prisma.

## Features
- User registration and login
- JWT authentication
- Protected routes
- Create moods for logged-in users
- Prisma ORM with migrations

## Tech Stack
- Node.js
- TypeScript
- Express
- Prisma
- PostgreSQL
- JWT

## Run Locally

```bash
npm install
npm run dev
